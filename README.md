# 📚 도서관 탐험대

규암초등학교 **꿈빛도서관** 이용자 교육 앱입니다.
학생들이 도서관 이용법을 재미있게 배우도록 만든 웹앱(PWA)이에요.

> 설치 프로그램도, 서버도, 앱스토어도 필요 없습니다.
> 브라우저로 열고 **"홈 화면에 추가"** 하면 앱처럼 쓸 수 있어요.

---

## 무엇을 배우나요

| 단계 | 내용 | 방식 |
|:---:|:---|:---|
| 1 | 도서관 소개 | 배치도 · 이용 시간 · 선생님 인사 |
| 2 | 도서관 용어 | 설명을 읽고 낱말 찾기 (짝 맞추기 게임) |
| 3 | 빌리고 반납하기 | 대출/반납 5단계 + 자가대출기 사용법 |
| 4 | 도서관 약속 | YES / NO 카드 게임 |
| 5 | 보물찾기 | 청구기호 읽는 법 + KDC 컬러맵 |
| 🏅 | 도전! 도서관 퀴즈 | 5문제 중 4문제 이상 맞히면 수료 배지 + 쿠폰 |

그 밖에 **반납일 D-Day 알림**, **읽어주기(TTS)**, **오프라인 열람**,
**사서 선생님용 쿠폰 검수 화면**이 있습니다.

---

## 실행 방법

### 그냥 보기
[index.html](index.html) 을 브라우저로 열면 됩니다.

> ⚠️ 파일을 더블클릭해서 열면(`file://`) **오프라인 기능만** 동작하지 않습니다.
> 나머지는 모두 정상입니다.

### 제대로 실행 (오프라인 기능 포함)
VS Code에서 **Live Server** 확장을 설치하고 `index.html` 우클릭 → *Open with Live Server*

### 배포
GitHub 저장소 → **Settings → Pages → Branch: main → Save**
몇 분 뒤 `https://dhsojo31-rgb.github.io/library/` 로 접속됩니다.

학생 안내:
- **iPhone**: Safari로 접속 → 공유 → "홈 화면에 추가"
- **Android**: Chrome으로 접속 → 메뉴(⋮) → "홈 화면에 추가"

## 내용을 고치고 싶다면

**[data.js](data.js) 파일 하나만 고치면 됩니다.** 다른 파일은 건드리지 마세요.

```js
const SCHOOL = {
  name: '규암초등학교',
  libraryName: '꿈빛도서관',
  librarian: '사서 선생님',   // 성함을 넣고 싶으면 '김○○ 선생님'
  ...
};

const LOAN_RULE = { maxBooks: 2, days: 7 };   // 대출 2권 / 1주
```

이용 시간, 공지사항, 퀴즈 문제, 낱말, 배치도 모두 이 파일 안에 있습니다.
따옴표 `' '` 안의 글자만 바꾸고 쉼표 `,` 는 지우지 마세요.

`🔴 TODO` 라고 적힌 곳은 아직 확인이 필요한 임시 정보입니다.

## 파일 구성

```
index.html      화면 뼈대 (모든 화면이 여기 있고 JS가 보여주고 숨김)
style.css       디자인
app.js          화면 전환 · 이벤트 처리
data.js         모든 콘텐츠 ⭐ 고칠 일이 있으면 여기
storage.js      진행률 · 배지 · 쿠폰 저장
barcode.js      바코드 그리기 (CODE 39 직접 구현, 외부 라이브러리 없음)
manifest.json   "홈 화면에 추가" 설정
sw.js           오프라인 캐싱
icons/          앱 아이콘
```

외부 라이브러리를 하나도 쓰지 않습니다. 순수 HTML / CSS / JavaScript 입니다.

## 알아둘 점

**사서 선생님용 화면은 이 기기의 데이터만 보여줍니다.**
서버가 없어서 학생 기록이 각자의 폰 안에만 저장되기 때문입니다.
전교생 현황 조회가 필요하면 Firebase 같은 서버 연동이 추가로 필요합니다.
현재 화면은 *"학생이 폰을 보여주면 선생님이 쿠폰을 사용 처리하는 검수 도구"* 로 설계했습니다.

**관리자 PIN은 보안 장치가 아닙니다.**
브라우저 개발자도구로 볼 수 있는 값이라, 실수로 들어가는 것을 막는 잠금 정도입니다.
